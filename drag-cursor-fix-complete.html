🔧 技术实现细节

1. 全局CSS覆盖

/* 强制覆盖所有元素的光标 */ body.dragging-active * { cursor: move !important; } body.dragging-detached * { cursor: copy !important; }

2. 动态类名管理

// 拖拽开始时添加类 document.body.classList.add('dragging-active'); // 进入分离模式时切换类 document.body.classList.remove('dragging-active'); document.body.classList.add('dragging-detached'); // 拖拽结束时清理类 document.body.classList.remove('dragging-active', 'dragging-detached');

3. dataTransfer配置优化

// 更兼容的设置方式 event.dataTransfer.effectAllowed = 'copyMove'; event.dataTransfer.dropEffect = 'move';
