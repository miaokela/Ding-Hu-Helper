拖拽分离功能

  1. 在WebView标签页上按下鼠标左键
  2. 拖拽标签页到软件窗口外部区域
  3. 松开鼠标，自动创建独立的分离窗口
  4. 分离窗口包含完整的WebView功能

智能合并功能

  1. 拖拽分离窗口靠近主窗口

  3. 显示合并提示和高亮效果

💡 实现原理

通过Electron的多窗口机制，结合HTML5拖拽API和IPC通信，实现标签页在不同窗口间的无缝迁移。 利用窗口位置监听和碰撞检测算法，智能识别用户的合并意图。

技术栈详情

• Electron多窗口管理
• IPC进程间通信
• HTML5 Drag & Drop API
• Vue 3响应式系统
• WebView状态保持
• 窗口碰撞检测算法
• 实时位置监听
• TypeScript类型安全

⚠️ 注意事项

此功能需要在Electron环境中运行。在普通浏览器中，会降级为使用window.open创建新窗口， 但无法实现智能合并功能。请确保在应用的Electron版本中测试此功能。
