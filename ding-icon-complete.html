// 以图标中心(256,256)为基准偏移

ing文字 (跟随对齐)

x="310" y="320"
font-size="65"
text-anchor="start"
// y值与D相同，实现底部对齐

🚀 使用说明

1. 选择图标风格

node select-icon.cjs

运行脚本，选择1-3中的任意风格

2. 生成PNG文件 (必需)

方法A: 打开 svg-to-png-converter.html 在线转换

方法B: 使用图像软件手动转换

方法C: ImageMagick: magick multi-browser-logo.svg multi-browser-logo.png

3. 生成macOS图标 (macOS用户)

./generate-macos-icon.sh

自动生成.icns文件

4. Windows用户检查

generate-windows-icon.bat

检查文件状态和生成报告
